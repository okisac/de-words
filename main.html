MemoryFlip

Diese Webanwendung hilft beim Lernen von deutschen und türkischen Wörtern:

  • Es werden neue Wortpaare hinzugefügt.
  • Die letzten 5 hinzugefügten Paare werden auf dem Bildschirm angezeigt.
  • Aus allen hinzugefügten Wörtern wird ein zufälliges Paar auf dem Bildschirm angezeigt.
  • Zuerst wird das deutsche Wort angezeigt. Der Benutzer versucht, die Bedeutung zu erraten.
  • Dann wird das türkische Wort angezeigt.
  • Dieser Vorgang wird wiederholt.

About

This web application helps you learn German-Turkish vocabulary:

  • New word pairs are added.
  • The last 5 pairs added are displayed on the screen.
  • A random pair is displayed on the screen from all the words added.
  • The German word is displayed first. The user tries to guess its meaning.
  • Then the Turkish word is displayed.
  • This process repeats.

Entwickelt mit HTML, CSS und JS. Backend mit Node.js, Datenbankmanagement über PostgreSQL auf Supabase. Veröffentlicht (Deployed) über Render.

Tech Stack

Developed with HTML, CSS, and JS. Backend with Node.js, database management via PostgreSQL on Supabase. Deployed via Render.
